## Podman Desktop

- web site : https://podman-desktop.io/

- windows installation → https://podman-desktop.io/docs/installation/windows-install

- macos installation → https://podman-desktop.io/docs/installation/macos-install

- Verify CLI: 
  
  ```
  podman --version
  ```

## Git (CLI)

- Git on Windows → https://git-scm.com/install/windows

- Git on Mac → https://git-scm.com/install/mac

- Verify CLI: 
  
  ```
  git --version
  ```
